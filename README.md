This project uses Doxygen for generating code documentation.

1. Make sure you have Doxygen and Graphviz installed on your system
2. Run the following command from the project root:
   doxygen Doxyfile
3. The documentation will be generated in the "docs/html" directory
4. Open "docs/html/index.html" in your web browser to see the generated documentation
